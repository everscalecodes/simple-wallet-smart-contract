# Installation
## Requirements
* [Node.js](https://nodejs.org) >= `16.x`
* [Yarn](https://classic.yarnpkg.com) >= `1.22.x`
* [Free TON Development Environment](https://github.com/tonlabs/tondev) >= `0.7.x`

## Set up project
```sh
yarn install
```

Copy `config.example.ts` to `config.ts`
```sh
yarn cp
```

Check and edit `config.ts`
